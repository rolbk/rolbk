Hi there, I'm Emanuel! Online I mostly go by **rolbk**. I'm a security researcher finishing my MSc in Cyber Security at ETH Zürich - this account collects my public research, tooling, and side projects.

### Security Research

- **Master's thesis (Google × ETH COMSEC):** a Spectre fuzzer built on a high-fidelity speculation detector for AMD Zen CPUs via custom microcode (EntrySign). Detects Spectre v1/v2, Retbleed, and Phantom, with an ongoing search for novel speculative-execution gadgets. Fixes from this work landed upstream in [google/security-research](https://github.com/google/security-research/pull/341) (zentool).
- **[AOSP-fuzzer-autogen](https://github.com/rolbk/AOSP-fuzzer-autogen):** automated fuzz-harness generation for Android system services (AIDL + libFuzzer), built with the [HexHive](https://hexhive.epfl.ch/) lab at EPFL.
- **CTF:** playing pwn with [0rganizers](https://org.anize.rs/) - including DEF CON CTF Finals 2025 in Las Vegas. Occasional writeups on [my blog](https://rolbk.com).

### Other Projects

- **[SCION](https://github.com/scionproto/scion):** merged upstream contributions (standalone daemon mode, library refactoring) from my work at ETH's Network Security Group.
- **[Kathará](https://github.com/KatharaFramework/Kathara):** merged PRs across the network-emulation framework - core, [Docker images](https://github.com/KatharaFramework/Docker-Images) (SCION support), and the [lab checker](https://github.com/KatharaFramework/kathara-lab-checker) (OSPF/SCION/HTTP checks) powering ETH's networking labs for 350+ students.
- **[BoostHut](https://github.com/rolbk/BoostHut):** Barnes-Hut N-body simulation, microarchitecturally optimized - cache-aware data layout, SIMD, and per-hotspot profiling.
- **[VitalVision](https://github.com/rolbk/VitalVision):** mobile platform for wearable ECG/PPG signal validation - my BSc thesis, written at ETH's SIPLAB.

Everything else - small tools, experiments, and archived history - lives in my [repositories list](https://github.com/rolbk?tab=repositories). My fully open-source, no-cloud smart home (including the electrically extendable sofa) lives over on [the blog](https://rolbk.com).
